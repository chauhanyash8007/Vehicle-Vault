🚗 Vehicle Vault – MERN Stack Vehicle Comparison System

Vehicle Vault is a full-stack web application developed using the MERN Stack (MongoDB, Express.js, React.js, Node.js) that allows users to search, filter, and compare vehicles based on their specifications, pricing, and features.

The platform helps users make informed vehicle purchasing decisions by providing side-by-side vehicle comparison, accessories recommendations, and system notifications. By analyzing similarities and differences between vehicles, users can easily evaluate multiple options before making a final buying decision.

📌 Project Overview
Vehicle Vault provides a centralized platform where users can:

🔎 Search vehicles using filters such as brand, model, price, and specifications
📊 Compare two vehicles side-by-side to analyze similarities and differences
📄 View detailed vehicle specifications and features
🧰 Explore recommended accessories for specific vehicles
🔔 Receive system notifications from the admin
👨‍💼 Admin panel to manage vehicles, accessories, and notifications

The system uses MongoDB collections designed through UML modeling and follows a structured backend architecture for scalable MERN stack development.
